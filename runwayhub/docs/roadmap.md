## 🤝 Contributing

### Wie können Sie beitragen?
1. **Issues:** Bugs melden
2. **PRs:** Features hinzufügen
3. **Docs:** Dokumentation verbessern
4. **Tests:** PHPUnit Tests schreiben

### Commit Conventions
```bash
feat: neue Features
fix: Bugfixes
docs: Dokumentation
style: Code Style
refactor: Refactoring
test: Tests
chore: Andere Änderungen
```

### Pull Request Requirements
- [ ] Code Style: PSR-12
- [ ] Tests: PHPUnit Tests
- [ ] Docs: Dokumentation aktualisieren
- [ ] Security: Security Check
- [ ] CI/CD: Alle Tests grün
