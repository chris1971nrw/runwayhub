## Deployment Commands

### Local Development
```bash
cd /runwayhub
php -S localhost:8080 -t public
```

### Production (GitHub Pages)
```bash
# Build static assets
npm run build

# Deploy to GitHub Pages
ghp-import build/
```

### Database Backup
```bash
sqlite3 runwayhub/database.sqlite ".backup /backups/runwayhub-$(date +%Y%m%d).sqlite"
sqlite3 users.sqlite ".backup /backups/users-$(date +%Y%m%d).sqlite"
```

### Health Check
```bash
# Test endpoints
curl http://localhost:8080/
curl http://localhost:8080/login.php
curl http://localhost:8080/weather-widget.html

# Check PHP syntax
find public -name "*.php" -exec php -l {} \;
```

## Troubleshooting

### Common Issues

**500 Internal Server Error**
- Check PHP error logs
- Verify database connection
- Ensure .env variables are set

**403 Forbidden**
- Check file permissions
- Verify .htaccess rules
- Ensure public/ is web-accessible

**Weather Data Not Loading**
- Check API keys
- Verify API endpoints
- Review rate limits

**Login Not Working**
- Verify database has pilot accounts
- Check session storage
- Review error logs
