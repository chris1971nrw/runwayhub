# Demo Agents

Autonome Agents für das Demo-System:
- FlightAutomator: Automatischer Flug-Scheduler
- PIREPGenerator: PIREP-Simulation
- BookingGenerator: Buchungs-Generator
- FleetStatusUpdater: Flotten-Status Aktualisierung
- IssuesWatcher: GitHub Issues Monitor
- DemoSystemSync: Sync mit Haupt-System
