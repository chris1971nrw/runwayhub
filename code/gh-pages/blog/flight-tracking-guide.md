## Introduction

Learn how to use RunwayHub's flight tracking features to monitor flights in real-time. Whether you're a passenger, pilot, or aviation enthusiast, this guide will help you get started.

## Getting Started

### 1. Access the Flight Tracker

Visit the main landing page and navigate to the flight tracking section. You'll see a live flight board displaying aircraft in real-time.

### 2. Search for a Flight

Use the search bar to look up any flight by:
- Flight number (e.g., LH400)
- Airport code (e.g., FRA)
- Airline name
- Aircraft registration

### 3. View Flight Details

Each flight display shows:
- Current position on map
- Altitude and speed
- Destination airport
- Estimated time of arrival
- Aircraft type
- Flight status (on time, delayed, etc.)

## Flight Status Indicators

- 🟢 **Green:** On time, en route
- 🟡 **Yellow:** Slight delay
- 🔴 **Red:** Significant delay
- ⚪ **Gray:** Scheduled but not yet active
- ❌ **Crossed out:** Cancelled flight

## Real-Time Updates

RunwayHub updates flight data every 60 seconds. Refresh the page or wait for automatic updates to see the latest information.

## Troubleshooting

**Flight not showing?**
- Check if flight exists
- Verify spelling of flight number
- Ensure you're on correct date
- Check regional restrictions

**Map not loading?**
- Refresh the page

- Check browser compatibility
- Verify internet connection
