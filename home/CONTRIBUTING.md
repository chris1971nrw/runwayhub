# Contributing zu RunwayHub

Danke für Ihr Interesse an der Entwicklung von RunwayHub!

## Wie Sie beitragen können

- **Bug Reports**: Erstellen Sie Issues auf GitHub
- **Features**: Vorschläge im Issue Tracker
- **Code**: Pull Requests nach dem Workflow
- **Dokumentation**: Verbesserungen an der Docs

## Entwicklungsumgebung

### Voraussetzungen

- PHP 8.2+
- MySQL 8.0+
- Composer
- Git

### Setup

```bash
# Klonen
git clone https://github.com/chris1971nrw/runwayhub.git
cd runwayhub

# Dependencies installieren
composer install

# Umgebung konfigurieren
cp .env.example .env
# .env bearbeiten
```

## Code Standards

- **PSR-12** für PHP Code
- **PHPUnit** für Tests
- **PHPStan** für Static Analysis

## Pull Request Workflow

1. Fork das Repository
2. Feature-Branch erstellen
3. Commits nach Conventions
4. Push zum Remote
5. Pull Request öffnen

## Review Process

- Code Reviews erforderlich
- Tests müssen bestehen
- Documentation aktualisieren

## Code of Conduct

Seien Sie respektvoll und konstruktiv.

## Fragen?

Erstellen Sie ein Issue oder schreiben Sie uns:
- Email: info@runwayhub.de
- GitHub Issues
